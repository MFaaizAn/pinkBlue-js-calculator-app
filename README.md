# pinkBlue-js-calculator-app
Pink and Blue JavaScript Calculator App

# 🧮 Simple Web Calculator

Sebuah aplikasi kalkulator berbasis web sederhana yang dibangun menggunakan HTML, CSS, dan JavaScript vanilla. Proyek ini dibuat untuk melatih pemahaman mengenai manipulasi DOM (Document Object Model) dan fungsionalitas dasar matematika di JavaScript.

## ✨ Fitur
* **Operasi Dasar Matematika:** Penjumlahan (`+`), Pengurangan (`-`), Perkalian (`*`), dan Pembagian (`/`).
* **Fitur Tambahan:** Perhitungan persen (`%`), tombol hapus satu karakter (`Del`), dan reset layar (`AC`).
* **Desain Responsif:** Layout tombol menggunakan CSS Grid yang rapi dan nyaman dilihat.

## 🛠️ Teknologi yang Digunakan
* **HTML5** - Untuk struktur semantik halaman dan elemen input/tombol.
* **CSS3** - Untuk styling layout menggunakan Flexbox, CSS Grid, dan kustomisasi warna (`hsl`).
* **JavaScript (ES6)** - Untuk logika kalkulasi, penanganan event klik, dan manajemen *state* display.
